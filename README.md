# Система бронирования и управления для гостевого дома (Flutter / Firebase)

Кроссплатформенное мобильное приложение (iOS/Android) для туристов и администраторов. Позволяет автоматизировать учет бронирований, исключить накладки (овербукинг) и вести учет финансов.

## ✨ Ключевые функции

### Для туриста:
* **Календарь занятости:** Просмотр свободных дат в реальном времени.
* **Бронирование:** Выбор домиков (включая комбо-объекты вроде «Вся база» или «Теремок целиком»).
* **Умная валидация:** Защита от бронирования на одну и ту же дату (система учитывает время заезда и выезда).

### Для администратора (Админ-панель):
* **Ручное создание броней:** Добавление бронирования прямо из календаря.
* **Управление жильем:** Редактирование описаний, цен и фотографий домиков в реальном времени.
* **Финансовая аналитика:** Графики доходности и статистика по каждому домику.

## 🛠 Технологический стек
* **Фреймворк:** Flutter (Dart)
* **База данных:** Firebase Firestore (синхронизация в реальном времени)
* **Архитектура:** Provider (State Management)
* **Графики:** FL Chart
